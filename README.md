# SentimenMBG

**SentimenMBG** adalah aplikasi web berbasis **Flask** yang dirancang untuk menganalisis opini publik terhadap **Program Makan Bergizi Gratis (MBG)** melalui komentar pada video YouTube. Sistem ini memungkinkan pengguna memasukkan URL video YouTube, mengambil komentar secara otomatis, melakukan preprocessing teks Bahasa Indonesia, lalu mengklasifikasikan sentimen komentar ke dalam tiga kategori: **positif**, **netral**, dan **negatif** menggunakan model machine learning yang telah dilatih.

Aplikasi ini juga menyediakan **dashboard visual interaktif** untuk menampilkan hasil analisis dalam bentuk ringkasan statistik, distribusi sentimen, pie chart, bar chart, dan word cloud. Dengan antarmuka modern berbasis **Bootstrap 5** dan proses analisis otomatis, SentimenMBG dapat digunakan sebagai alat bantu untuk memahami persepsi masyarakat terhadap implementasi Program Makan Bergizi Gratis secara lebih cepat dan terstruktur.

## Fitur Utama

* **Analisis sentimen komentar YouTube** berdasarkan URL video yang dimasukkan pengguna
* **Klasifikasi 3 kelas sentimen**: Positif, Netral, dan Negatif
* **Preprocessing teks Bahasa Indonesia**, meliputi normalisasi kata, stopword removal, dan stemming
* **Visualisasi hasil analisis** dalam bentuk statistik, progress distribution, pie chart, bar chart, dan word cloud
* **Antarmuka web responsif** dengan tampilan modern dan user-friendly

## Teknologi yang Digunakan

* **Python**
* **Flask**
* **Bootstrap 5**
* **Chart.js**
* **Machine Learning / NLP**
* **YouTube Comment Scraper**

## Tujuan Pengembangan

Project ini dikembangkan untuk membantu pengguna dalam memahami kecenderungan opini publik terhadap Program Makan Bergizi Gratis berdasarkan komentar masyarakat di platform YouTube. Selain itu, sistem ini juga dapat menjadi implementasi penerapan **Natural Language Processing (NLP)** dan **machine learning** pada analisis sentimen berbahasa Indonesia.
